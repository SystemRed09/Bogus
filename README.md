# <span style="color:#9949CC">IT</span>Tech Classroom Printer Checker
### Quick ip4 checker for printers on the KSU Tech Classroom checklist
A Simpler way to check all the printers on IC and UC routes for the Kansas State University main campus. Simply download the files and run the .sh/.bats to generate the status of the printers. For those needing more control or would like to test/modify their copies, the exe accepts custom arguments that can be found by running the .exe directly with no arguments. `main.py` Features the code for the arguments as well. 

For IT Tech Class workers: Grab the files from the onedrive to ensure you have the correct default files for the printers. The GitHub page does not support these for security.
